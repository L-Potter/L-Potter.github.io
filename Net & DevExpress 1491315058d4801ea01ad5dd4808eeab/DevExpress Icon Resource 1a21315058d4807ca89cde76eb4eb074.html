DevExpress Icon Resource

在 DevExpress 內建圖庫 (Image Collection) 中，有許多可用的 內建圖標 (Icons)，你可以將這些圖標添加到 Properties.Resources，然後在 程式中使用。

✅ 方法 1：從 DevExpress 圖庫手動導出並加入 Properties.Resources

  1. 開啟 DevExpress 圖片庫
    • 在 Visual Studio 中，開啟 "DevExpress Image Picker"
    • Tools > DevExpress > Local Image Picker
    • 或者在 DevExpress 設計器 (如 RibbonControl) 的 Image 屬性 選擇 ...。
  1. 選擇圖片並匯出
    • 找到你需要的圖標 (如 "Save" 或 "Print"）。
    • 點擊 "Save As"，將圖標匯出為 PNG。
  1. 將圖標加入 Properties.Resources
    • 開啟 Resources.resx（在 Solution Explorer 裡 Properties > Resources.resx）。
    • 拖曳 PNG 圖片 到 Resources 窗口，或點擊 Add Resource > Add Existing File 選擇圖片。
  1. 在程式中使用
    btnSave.Glyph = Properties.Resources.icon_save;

✅ 方法 2：直接使用 DevExpress 內建圖標

如果你不想手動匯出圖標，DevExpress 允許你直接使用內建圖標 而不需要加入 Properties.Resources。

📌 在程式中使用內建圖標

using DevExpress.Images;
using DevExpress.XtraBars;

private void Form1_Load(object sender, EventArgs e)
{
    // 建立 BarButtonItem
    BarButtonItem btnSave = new BarButtonItem();
    btnSave.Caption = "Save";

    // 直接使用 DevExpress 內建圖庫
    btnSave.Glyph = ImageResourceCache.Default.GetImage("images/save/save_16x16.png");

    // 加入按鈕到 Ribbon
    ribbonPageGroup1.ItemLinks.Add(btnSave);
}

🔹 ImageResourceCache.Default.GetImage("images/save/save_16x16.png")

  • save_16x16.png 是 DevExpress 內建的小圖標。
  • 你可以使用 32x32、48x48 來調整大小。

✅ 如何找到 DevExpress 內建圖標名稱？

你可以在 DevExpress 圖庫瀏覽器 中找到圖標：

  1. 打開 DevExpress 圖庫瀏覽器Tools > DevExpress > Image Picker
  1. 查看圖標的完整名稱
    例如：
    • images/actions/add_16x16.png
    • images/save/save_32x32.png
    • images/print/print_32x32.png
  1. 在程式中使用 ImageResourceCache.Default.GetImage() 來載入圖標

🚀 結論

✔ 方法 1：手動匯出圖標到 Properties.Resources，然後使用 Properties.Resources.icon_save。

✔ 方法 2：直接使用 ImageResourceCache.Default.GetImage() 動態載入 DevExpress 內建圖標，更靈活，不需要手動管理資源。

建議 如果 DevExpress 內建圖標夠用，直接用 方法 2 比較方便！🚀🎨

如果要使用 Properties.Resources.icon_save 來加載圖片，則 Build Action 應該設定為 Resource，因為 DevExpress 使用的是 .resx 內的資源管理。

✅ 設定 Build Action

  1. 將圖片 (icon_save.png) 加入專案
    • 在 Solution Explorer 裡，右鍵點擊專案名稱，選擇 Add > Existing Item…。
    • 選擇你要的圖片（如 icon_save.png），然後點擊 Add。
  1. 開啟圖片的 Properties
    • 在 Solution Explorer 中，找到 icon_save.png。
    • 右鍵點擊圖片，選擇 Properties。
  1. 設定 Build Action 為 Resource
    • 在 Properties 視窗中，找到 Build Action。
    • 設定 Build Action: Resource。

✅ 在程式中使用

btnSave.Glyph = Properties.Resources.icon_save;

這樣，圖片將被 嵌入 (Embedded Resource) 到 Resources.resx 中，並且可以透過 Properties.Resources.icon_save 直接存取。

🚀 其他選項 (Build Action) 的差異

Build Action用途
None檔案不會包含在編譯輸出中
Content檔案會包含在輸出中，但不會嵌入到 DLL
Embedded Resource內嵌到 DLL，但無法透過 Properties.Resources 直接存取，需要 Assembly.GetManifestResourceStream()
Resource內嵌到 DLL 並且可以透過 Properties.Resources.icon_save 存取（這是最推薦的設定！）

✅ 結論

✔ 如果要透過 Properties.Resources.icon_save 存取圖片，請設定 Build Action 為 Resource。

✔ 如果想要手動載入內嵌圖片，可以使用 Embedded Resource，但需要 Assembly.GetManifestResourceStream()。

🚀 Resource 是最方便的選擇！ 🎨

In addition to using resource files, you can directly embed your icon as an embedded resource. Here’s how:

  1. Add the Image to Your Project:
  • In Solution Explorer, right-click on your project, then select Add > Existing Item….

  • Navigate to the image file (e.g., spiral.png), select it, and click Add.
  1. Set the Build Action:
  • In Solution Explorer, locate the added image file.
  • Right-click on the file, select Properties, and set the Build Action property to Resource.

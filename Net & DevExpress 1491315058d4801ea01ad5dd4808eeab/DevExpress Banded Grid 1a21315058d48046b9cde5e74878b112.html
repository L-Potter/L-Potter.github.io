DevExpress Banded Grid

WinForms Grid - Banded Grid View Basics

Grid Designer 為DevExpress專屬designer.cs UI

 Bands 提供多種分類方式來顯示Grid Col

在 DevExpress Grid Designer（或者直接在 Visual Studio 設計器）內 新增 GridBand 到 BandedGridView 時，Designer.cs 會自動產生對應的代碼來初始化 GridBand，並設定其基本屬性，例如：

  • Caption（顯示標題）
  • Name（唯一識別名稱）
  • VisibleIndex（顯示順序）
  • Width（寬度）

而後，在 Form.cs 內，你可以手動為 GridBand 動態新增 BandedGridColumn。

✅ Designer.cs 自動生成的代碼

如果你在 GridControl 的 BandedGridView 內新增一個 GridBand（假設名稱為 gbBuddy），Form1.Designer.cs 會自動產生：

private void InitializeComponent()
{
    this.gridControl1 = new DevExpress.XtraGrid.GridControl();
    this.bandedGridView1 = new DevExpress.XtraGrid.Views.BandedGrid.BandedGridView();
    this.gbBuddy = new DevExpress.XtraGrid.Views.BandedGrid.GridBand();

    ((System.ComponentModel.ISupportInitialize)(this.gridControl1)).BeginInit();
    ((System.ComponentModel.ISupportInitialize)(this.bandedGridView1)).BeginInit();

    // 設定 GridControl
    this.gridControl1.MainView = this.bandedGridView1;
    this.gridControl1.Dock = System.Windows.Forms.DockStyle.Fill;
    this.Controls.Add(this.gridControl1);

    // 設定 BandedGridView
    this.bandedGridView1.Bands.AddRange(new DevExpress.XtraGrid.Views.BandedGrid.GridBand[] {
        this.gbBuddy});
    this.bandedGridView1.GridControl = this.gridControl1;

    // 設定 GridBand（gbBuddy）
    this.gbBuddy.Caption = "Buddy 資料";
    this.gbBuddy.Name = "gbBuddy";
    this.gbBuddy.VisibleIndex = 0;
    this.gbBuddy.Width = 200;

    ((System.ComponentModel.ISupportInitialize)(this.gridControl1)).EndInit();
    ((System.ComponentModel.ISupportInitialize)(this.bandedGridView1)).EndInit();
}

✅ 在 Form.cs 中新增 BandedGridColumn

GridBand 本身不會顯示資料，你需要在 .cs 檔案內手動將 BandedGridColumn 加入 GridBand：

using DevExpress.XtraGrid.Views.BandedGrid;

private void Form_Load(object sender, EventArgs e)
{
    // 新增 BandedGridColumn
    BandedGridColumn colBuddyName = new BandedGridColumn();
    colBuddyName.Caption = "Buddy 名稱";
    colBuddyName.FieldName = "BUDDY_NAME";
    colBuddyName.Visible = true;

    // 加入到 GridBand
    gbBuddy.Columns.Add(colBuddyName);

    // 加入到 BandedGridView
    bandedGridView1.Columns.Add(colBuddyName);
}

🔹 說明

  • BandedGridColumn 是 屬於 BandedGridView 的欄位，需要手動新增。
  • gbBuddy.Columns.Add(colBuddyName); → 這樣欄位才會歸屬於 GridBand。
  • bandedGridView1.Columns.Add(colBuddyName); → 確保 BandedGridView 管理此欄位。
  • FieldName = "BUDDY_NAME" → 必須對應你的 DataSource 欄位名稱。
